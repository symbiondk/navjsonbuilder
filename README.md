# navjsonbuilder
A simple interface for building a navigation hierarchy in JSON and output in bootstrap navbar HTML


How to use:
1)
Edit the navigation via index.php in your browser.

2a) Use a template:
Use php include and point it to nav.navbar_bootstrap.php in your project. This will output nav-items for your navbar.

2b) Render your own HTML:
Fetch the rendered JSON from nav.json and use it to build your own HTML syntax.
